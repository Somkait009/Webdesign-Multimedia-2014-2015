Informationen

Der interaktive CSS-Simulator verdeutlicht die Funktionsweise der CSS-Dateien. Durch Änderung der Werte passt sich die Seite sofort in Echtzeit den neuen Einstellungen an.

Änderungen der Klasse "HTML" wirken sich auf die gesamte Seite aus.
Änderungen der Klasse "Body" wirken sich auf den Rumpf der Seite aus.
Änderungen der Klasse "Title" wirken sich auf die Überschrift der Seite aus.
Änderungen der Klasse "Style" wirken sich auf den CSS Editor aus.

color: blue | aqua | #123456 | #565
font-family: "Lato" | "Source Code Pro" | FontAwesome
font-size: 1em | 20px | 100%
margin: 3em | 50px
