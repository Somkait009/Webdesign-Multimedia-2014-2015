Der interaktive CSS-Simulator verdeutlicht die Funktionsweise der CSS-Dateien.
Durch Änderung der Werte passt sich die Seite sofort in Echtzeit den neuen Einstellungen an.

Änderungen der Klasse "HTML" wirken sich auf die gesamte Seite aus. Änderungen der Klasse "Body" wirken sich auf den Rumpf der Seite aus. Änderungen der Klasse "Title" wirken sich auf die Überschrift der Seite aus. Änderungen der Klasse "Style" wirken sich auf das Feld aus, in welcher die CSS-Dateien angezeigt werden.

Mögliche Eingaben:

Farbe: blue | aqua | #123456 | #565
Schriftart: "lato" | "Source Code Pro" | FontAwesome
Textgröße: 1em | 20px | 100%
Abstand: 3em | 50px
