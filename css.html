CSS

Entwicklung

CSS: "Cascading Style Sheet" ist eine Beschreibungssprache für Stylesheets. Sie dient als "Formatvorlage" für HTML-Seiten und wurde im Dezember 1996 vom W3C herausgegeben. Der Vorgänger von CSS war "Cascading HTML Style Sheet" (CHSS), welcher im Oktober 1994 von Håkon Wium Lie vorgeschlagen wurde.

CSS vs HTML

Der Style der Seite kann in reinem HTML wie folgt festgelegt werden:

                    <html>
    ...
    <body bgcolor="black">
        ...
        <div bgcolor="red">...</div>
        ...
        <div bgcolor="red">...</div>
        ...
        <div bgcolor="red">...</div>
        ...
    </body>
</html>

Der Style kann durch die Kombination aus HTML und CSS wie folgt festgelegt werden:

                    <html><head>
    ...
    <style>
        body {
            background-color:black;
        }

        div {
            background-color:red;
        }
    </style></head><body>
    ...
    <div>...</div> ...
    <div>...</div> ...
</body></html>

Link-Tag

Der Link-Tag verweist auf eine CSS Datei, die von mehreren HTML-Seiten gleichzeitig genutzt werden kann. Dadurch spart man sich Speicherplatz und Ladezeit.

<link rel="stylesheet" type="text/css"
href="config/style.css" media="screen">

Hierbei wird eine logische Dateibeziehung (link = Verweis, rel = relationship) auf das "stylesheet" erstellt. Dies gibt den Bezug zu einer Datei (style.css) mit Style-Definitionen, die zur Darstellung herangezogen werden.

Medien

Externe Dateien, z.B. Bilder, sollten möglichst relativ referenziert werden. Relative Pfade suchen in dem Ordner, in dem die Quelldatei liegt und orientieren sich an dessen Struktur. Anstatt eines absoluten Pfades, wie z.B. "C:\Users\...\images\bild.jpg" sollte ein relativer Pfad, wie "images/bild.jpg" verwendet werden. Dieser findet das Bild unabhängig vom Pfad des Ordners.

Bilder werden mit dem HTML-Tag <img> eingebunden. Das Bild "logo.png" wird im Verzeichnis "images" eingebunden. Das Attribut "alt" wird benötigt, falls das Bild nicht vorhanden ist. In diesem Fall kann man ein Fehlerbild oder einen Fehlertext darstellen.
<img src="images/logo.png" alt="Logo">
Weitere Informationen zu Attributen von Bildern

Links können unter HTML wie folgt verwendet werden:
<a href="https://www.google.de/">Google</a>

Bei einzelnen Punkten innerhalb eines Dokuments spricht man von "Ankern":
<a id="Anker"></a>
Hiermit definiert man einen Anker mit der ID "Anker". Dieser kann wie folgt referenziert werden:
<a href="#Anker">Klick mich!</a>
